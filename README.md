Análise de Sentimentos com NLP

## 📌 Sobre o Projeto

Projeto desenvolvido durante a disciplina de Inteligência Artificial na UNDB com o objetivo de classificar automaticamente avaliações de filmes como positivas ou negativas utilizando Processamento de Linguagem Natural (NLP).

## 💡 Problema

A análise manual de milhares de comentários e avaliações é lenta e pouco escalável. Técnicas de NLP permitem automatizar esse processo.

## 🎯 Objetivos

- Processar textos.
- Aplicar técnicas de NLP.
- Classificar sentimentos automaticamente.
- Avaliar o desempenho do modelo.

## 📊 Dataset

### Nome
IMDb Dataset of 50K Movie Reviews

### Fonte
https://www.kaggle.com/datasets/lakshmi25npathi/imdb-dataset-of-50k-movie-reviews

### Descrição

O dataset contém aproximadamente 50 mil avaliações de filmes classificadas como positivas ou negativas.
